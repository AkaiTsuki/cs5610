This page contains some basic use of Jquery UI dialog.

Purpose

The first experiment is about create a dialog by clicking the button. This UI is using jquery-ui library and using the theme of ui-lightness. In additon to the popup behavior, I added default animate effect on dialog showing and hiding. And when the dialog show up, the back content will unable to react user's request until the dialog is closed.

Reference

jQuery UI 1.8- The User Interface Library for jQuery
