Purpose

In this experiment, I use the technique that the previous four experiments use to create a three column fluid layout. When the browser's width is larger than 768px, the page will be 3-column layout. When the width between 480px and 767px, it will be 2-column layout. And if the width is less than 480px, it will be single column.

By using fluid layout, the page can be viewed under devices that has different resolution.

Device Resolution Layout
Mobile portrait 320x480 1 column
Mobile landscape 480x320 2 column
Small tablet portrait 600x800 2 column
Small tablet landscape 800x600 3 column
Tablet portrait 768x1024 3 column
Tablet landscape 1024x768 3 column
